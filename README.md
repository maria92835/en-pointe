# En Pointe 🩰

> Your beginner's guide to ballet

En Pointe is a beginner-friendly ballet web app that helps new dancers learn the foundations of ballet in a fun, interactive way. Users can explore a library of classic ballet poses, read descriptions of each position, and follow along with an animated ballerina.

Built with React, Tailwind CSS, and a love for dance — En Pointe makes ballet accessible to anyone, no studio required.

## ✨ Features (some are not there yet!!)
- 🩰 Interactive pose library with descriptions
- 💃 Animated ballerina character
- 🎨 Clean, beginner-friendly UI
- 📱 Works in any browser

## 🛠 Built With
- [React](https://react.dev)
- [Vite](https://vitejs.dev)
- [Tailwind CSS](https://tailwindcss.com)

## 🏃 Run Locally
```bash
git clone https://github.com/maria92835/en-pointe.git
cd en-pointe
npm install
npm run dev
```
